== README

"This is the finance-tracker application"
1.Authentication System.
2.Users can track stocks, upto 10 per user.
3.Users can search for stocks using a search bar.
4.Users can choose to add a stock from results of search bar to their portfolio.
5.Users can look for friends, or other users, by name and email
6.Users can view portfolio of stocks their friends are tracking for investing needs.
7.Must be mobile friendly, so the styling has to be responsive.
